Java——JDBC执行SQL语句的两种方式

2017年5月4日

  本文记录一下JDBC执行SQL语句的两种方式，分别是使用Statment对象和使用PreparedStatment对象。

  使用JDBC前首先要在项目中导入数据库驱动jar包，本文使用的是MySQL，IDE使用的是Intellij IDEA，导入驱动的过程在此不赘述。
  使用JDBC执行SQL语句共分为三步：
    1. 获得连接类Connection对象
    2. 获得语句对象
    3. 执行语句
  其中使用PreparedStatment对象执行SQL语句更为灵活，可以传入参数。本文编写了一个查询数据库的示例。
